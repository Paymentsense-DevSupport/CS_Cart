Introduction

The files included in this pack are designed to integrate the PaymentSense Gateway with CSCart. This will enable you to connect your shopping cart to the PaymentSense gateway quickly and effectively.


NOTE: You must have installed CS-Cart version 4 before working through this integration document.


Integration

NOTE: You need to INSERT the SQL statements from within the paymentsense_sql.txt page into your CS-Cart database before continuing past this point.

NOTE: Using FTP, upload the three folders inside the files folder, 'app', 'design' and 'var' to the root of the CS-Cart environment this will place the files in the correct location.


Enable and Configure the PaymentSense Gateway

NOTE: All gateway configurations are carried out from within the admin section of your CS-Cart account.


NOTE: Make sure you have the GATEWAY MerchantID, Password and PreSharedKey
NOT THE LOGIN DETAILS FOR THE MERCHANT MANAGEMENT SYSTEM.


Add New Payment Method

  1. On the top right hand-side of the browser window you need to click on the Administration header and select Payment Methods.
  2. To the left hand side click on the + button to add a new icon, this will open a pop-up window.
  3. Insert a Name for your payment method. (for example "Pay by Card")
  4. If you have multiple stores select the store you wish to apply PaymentSense as a payment method to.
  5. Select PaymentSense for the Processor.
  6. You do not need to worry about the Template as this will be set automatically.
  7. Select the User Groups that you want to allow to use this payment method at checkout. (By default this is set to All).
  8. Enter a Description for your payment method.(for example "Secured By PaymentSense")
  9. Enter a Surcharge if you wish to charge your customers for using a card payment.
  10. Set a Surcharge Title for example (card handling).
  11. Set Payment Instructions for your customers.
  12. Ensure the Status is set as Active for the PaymentSense Account.
  13. set an icon.
  14. Click Create in the botom right corner.


Configure PaymentSense

  1. Click on the payment method you have just added.
  2. Click on Configure tab.
  3. Enter the Gateway Merchant ID.
  4. Enter your Pre-Shared Key.
  5. Enter the Gateway Merchant Password.
  6. Select your Transaction Type. If you take payments immediately then select Sale. If you deal with pre-orders and wish to take the payment manually at a later date then select Preauth.
  7. Select the default Currency
  8. Select Yes for CVV2 Required.
  9. Select Yes for Country Required.
  10. Select Yes for State/Province Required.
  11. Select Yes for City Required.
  12. Select Yes for Address Required.
  13. Select Yes for Zip/postal code Required.
  14. Click on Save

Your payment method will now be set up. Please attempt to run some test transactions.
